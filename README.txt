Contained are the files for Softscape's capstone project.

overview:
   This webapp is used for students to easily view syllabi for the CS department. It is also used so faculty members can easily update and change syllabi from semester to semester    to be approved by the course coordinators. 
 
sms-backend:
  Javascript and CSS
  uses nodejs for the backend framework
  
sms-homepage:
  Javascript and CSS
  uses reactjs 
  This is the default page for the website
  It contains a search bar for students to view syllabyi
  It also contains a link for faculty to log in and make changes
  This page contains the links for the Help, Terms&Conditions and About pages
